# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases.

## Tools Used
- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Business Problem
How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing and product strategies?

## Project Workflow
1. Data Cleaning using Python
2. SQL Analysis using PostgreSQL
3. Dashboard Development in Power BI
4. Business Recommendations

## Key Insights
- Female customers generated slightly higher revenue.
- Subscribers spent more than non-subscribers.
- Express shipping customers spent more per transaction.
- Loyal customers contributed significantly to revenue.

## Files Included
- Jupyter Notebook
- SQL Queries
- Power BI Dashboard
- Project Report
- Presentation.
